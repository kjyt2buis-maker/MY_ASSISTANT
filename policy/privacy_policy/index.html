MY_ASSISTANT 개인정보 처리방침

시행일: 2026년 5월 16일 | 최종 수정일: 2026년 5월 16일

MY_ASSISTANT(이하 "본 앱")의 개발자(이하 "개발자")는 정보주체의 자유와 권리 보호를 위해 「개인정보 보호법」 및 관계 법령이 정한 바를 준수하여, 적법하게 개인정보를 처리하고 안전하게 관리하고 있습니다.

이에 「개인정보 보호법」 제30조에 따라 정보주체에게 개인정보의 처리와 보호에 관한 절차 및 기준을 안내하고, 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

1. 개인정보의 처리 목적

개발자는 다음의 목적을 위하여 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 「개인정보 보호법」 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

1) 알림 기반 AI 브리핑 서비스 제공

이용자가 선택한 앱의 알림 메시지를 수집·분석하여, 인공지능(AI) 기술을 기반으로 주요 사건을 요약한 브리핑을 생성·제공하기 위함

2) 자연어 명령 처리 서비스 제공

이용자의 텍스트 또는 음성 입력을 분석하여, AI 기반으로 앱 내 데이터 조회·수정 등 명령을 처리하고 결과를 제공하기 위함

3) 앱 기능 제공 및 서비스 운영

이용자의 인간관계 관리, 대화방 관리, 개인 상태 기록 등 앱의 핵심 기능을 제공하기 위함

2. 처리하는 개인정보의 항목

개발자는 「개인정보 보호법」에 따라 서비스 제공을 위해 필요 최소한의 범위에서 개인정보를 처리합니다. 본 앱은 별도의 회원가입 절차가 없으며, 이용자의 동의를 받지 않고 처리하는 개인정보는 없습니다.

※ 14세 미만 아동 관련 안내: 본 앱은 14세 미만 아동을 대상으로 하지 않습니다. 14세 미만 아동이 본 앱을 이용하고자 하는 경우, 법정대리인의 동의가 필요하며, 법정대리인의 동의 없이 아동의 개인정보가 수집된 사실을 알게 된 경우 해당 정보를 지체없이 파기합니다.

4. 개인정보의 파기 절차 및 방법

① 개발자는 개인정보 보유기간의 경과, 처리 목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체없이 해당 개인정보를 파기합니다.

② 개인정보 파기의 절차 및 방법은 다음과 같습니다.

파기 절차

  • 이용자가 앱을 삭제(언인스톨)하면 단말기에 저장된 모든 데이터가 Android 운영체제에 의해 자동으로 삭제됩니다.
  • 이용자가 Android 설정에서 앱 데이터를 초기화하면 동일하게 모든 데이터가 삭제됩니다.
  • 앱 내에서 API 키 삭제, 개별 사건 카드 삭제, 멤버 비활성화 등 개별 삭제 기능을 제공합니다.

파기 방법

  • 전자적 파일: 암호화된 데이터베이스(SQLCipher)에서 해당 레코드를 삭제하며, 데이터베이스 파일은 앱 삭제 시 운영체제에 의해 완전히 제거됩니다.
  • 암호화 키: EncryptedSharedPreferences에 저장된 API 키 및 데이터베이스 암호화 키는 앱 삭제 시 함께 제거됩니다.

본 앱은 android:allowBackup="false" 설정을 적용하여 Android 자동 백업을 통한 개인정보 유출을 방지합니다.

5. 개인정보 처리업무의 위탁

개발자는 원활한 AI 기반 서비스 제공을 위하여 다음과 같이 개인정보 처리 업무를 위탁하고 있습니다.

위탁받는 자 (수탁자) 위탁업무
Google LLC
(Google AI / Gemini API)		 이용자의 알림 데이터 분석 및 브리핑 생성, 자연어 명령 처리를 위한 AI 응답 생성

이용자의 데이터가 Gemini API로 전송되기 전에, 본 앱은 개인정보 보호를 위해 다음과 같은 마스킹(비식별화) 처리를 수행합니다.

  • 이름 마스킹: 발신자·멤버 등 인물 이름을 [NAME_1], [NAME_2] 등의 식별 불가능한 코드로 대체
  • 대화방명 마스킹: 대화방 이름을 [ROOM_1], [ROOM_2] 등의 코드로 대체
  • 민감정보 마스킹: 전화번호, 계좌번호, 카드번호, 주민등록번호, 이메일 주소 등을 자동 탐지하여 마스킹(삭제) 처리

이러한 마스킹 매핑 정보는 메모리에서만 유지되며, 디스크에 저장되지 않고 각 세션 종료 시 즉시 삭제됩니다.

※ 참고: Gemini API로 전송된 데이터는 API 요청 처리 후 개발자의 서버에 별도 저장되지 않습니다. Google의 데이터 처리에 관한 상세 사항은 Google AI 서비스 이용약관 및 Google 개인정보처리방침을 참고하시기 바랍니다.

6. 개인정보의 국외 이전

개발자는 AI 브리핑 생성 및 자연어 명령 처리를 위해 이용자로부터 수집한 개인정보를 아래와 같이 국외에 이전하고 있으며, 「개인정보 보호법」 제28조의8제2항에 따라 다음과 같이 안내합니다.

항목 내용
이전받는 자 Google LLC (개인정보 보호 담당: privacy@google.com)
이전 국가 미국(USA) 등 Google 서버 소재국
이전 항목 마스킹(비식별화) 처리된 알림 메시지, 이용자 텍스트 명령, 첨부 이미지·파일(Base64 인코딩), AI 시스템 프롬프트
이전 목적 AI 기반 브리핑 생성 및 자연어 명령 처리 (처리위탁)
이전 시기 및 방법 이용자가 브리핑 생성 또는 명령을 요청할 때, TLS 암호화 통신을 통해 전송
보유 및 이용 기간 API 요청 처리 완료 후 개발자 측에서는 별도 저장하지 않음 (Google의 데이터 처리 정책에 따름)

국외 이전을 원치 않을 경우, 앱을 삭제하거나 API 키를 등록하지 않으면 됩니다. 다만 이 경우 AI 브리핑 및 자연어 명령 처리 기능을 이용할 수 없습니다.

7. 개인정보의 안전성 확보조치

개발자는 개인정보의 안전성 확보를 위해 다음과 같은 기술적 조치를 취하고 있습니다.

  • 데이터베이스 암호화: 모든 개인정보는 SQLCipher(AES-256)로 암호화된 데이터베이스에 저장됩니다.
  • API 키 암호화 저장: 이용자가 입력한 Gemini API 키는 Android EncryptedSharedPreferences(AES256_GCM)를 사용하여 암호화 저장됩니다.
  • 전송 구간 암호화: Gemini API와의 모든 통신은 HTTPS(TLS)를 통해 암호화되며, 평문(HTTP) 통신은 차단되어 있습니다.
  • 개인정보 비식별화: AI API 전송 전 이름·전화번호·계좌번호·주민등록번호·이메일 등을 자동 마스킹하여 원본 개인정보의 외부 전송을 최소화합니다.
  • 백업 차단: Android 자동 백업 기능을 비활성화(allowBackup="false")하여 백업을 통한 데이터 유출을 방지합니다.
  • 로그 보호: 릴리즈 빌드에서는 민감한 개인정보가 포함될 수 있는 상세 로그를 제거하여 로그를 통한 정보 노출을 방지합니다.

8. 자동화된 결정에 관한 사항

본 앱은 이용자의 알림 데이터를 AI가 분석하여 사건의 중요도·카테고리·긴급도 등을 자동으로 판정하는 기능을 포함하고 있습니다.

항목 내용
자동화된 결정의 사실 및 목적 수집된 알림 메시지를 AI(Gemini)가 분석하여 사건 요약, 중요도(critical/major/minor/trivial), 카테고리 분류를 자동으로 수행합니다. 이는 이용자에게 효율적인 정보 브리핑을 제공하기 위한 것입니다.
사용되는 주요 개인정보 마스킹 처리된 알림 메시지 내용, 대화방 유형 및 참여자 수, 이용자가 등록한 인간관계 정보
처리 절차 ① 알림 수신 → ② 개인정보 마스킹 → ③ AI 분석 요청 → ④ 중요도·카테고리 판정 결과 수신 → ⑤ 이용자에게 브리핑 카드 형태로 제공
정보주체의 권리 이용자는 AI가 판정한 사건의 중요도·카테고리 등에 대해 피드백을 제공하거나, 사건 카드를 직접 수정·삭제할 수 있습니다. 또한 브리핑 기능 자체를 사용하지 않을 수 있습니다.

본 앱의 자동화된 결정은 이용자에 대한 법적·재무적 효력을 가지지 않으며, 정보 정리 및 편의 제공 목적으로만 활용됩니다.

9. 정보주체와 법정대리인의 권리·의무 및 행사방법

① 정보주체는 개발자에 대해 언제든지 개인정보 열람·정정·삭제·처리정지 등을 요구할 수 있습니다.

② 본 앱은 모든 데이터를 이용자의 단말기에만 저장하므로, 이용자는 다음의 방법으로 직접 권리를 행사할 수 있습니다.

  • 개인정보 열람: 앱 내 각 화면(멤버, 대화방, 브리핑, 설정 등)에서 저장된 정보를 직접 확인할 수 있습니다.
  • 개인정보 정정·삭제: 앱 내에서 멤버 정보 수정·삭제, 대화방 메모 수정, 사건 카드 삭제, API 키 삭제 등을 직접 수행할 수 있습니다.
