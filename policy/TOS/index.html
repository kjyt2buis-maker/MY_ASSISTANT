My Assistant 이용약관

시행일: 2026년 5월 16일 | 최종 수정일: 2026년 5월 16일

본 이용약관(이하 "본 약관")은 My Assistant(이하 "본 앱") 개발자(이하 "개발자")가 제공하는 모바일 애플리케이션 서비스의 이용 조건 및 절차, 개발자와 이용자의 권리·의무 및 책임사항 등을 규정함을 목적으로 합니다.

본 앱을 설치하거나 이용하는 경우, 이용자는 본 약관의 내용을 충분히 읽고 이해하였으며 이에 동의한 것으로 간주합니다.

제1조 (목적 및 정의)

① 본 약관은 개발자가 제공하는 "My Assistant" 앱 서비스(이하 "서비스")의 이용에 관한 조건과 절차, 개발자와 이용자 간의 권리·의무 및 책임사항을 규정함을 목적으로 합니다.

② 본 약관에서 사용하는 용어의 정의는 다음과 같습니다.

  1. "서비스"란 개발자가 본 앱을 통해 제공하는 알림 기반 AI 브리핑 생성, 자연어 명령 처리, 인간관계·대화방 관리 등 일체의 기능을 말합니다.
  2. "이용자"란 본 약관에 따라 본 앱을 설치하고 서비스를 이용하는 자를 말합니다.
  3. "AI 기능"이란 외부 AI 서비스(Google Gemini API 등)를 활용하여 알림 분석, 브리핑 생성, 자연어 명령 처리 등을 수행하는 기능을 말합니다.
  4. "API 키"란 이용자가 AI 기능을 사용하기 위해 직접 발급받아 본 앱에 등록하는 Google Gemini API 인증 키를 말합니다.
  5. "알림 데이터"란 이용자가 알림 접근 권한을 허용한 앱으로부터 수신되는 알림에 포함된 발신자 이름, 메시지 내용, 대화방 정보 등을 말합니다.

제2조 (약관의 효력 및 변경)

① 본 약관은 본 앱 내 또는 개발자의 웹페이지를 통해 공지함으로써 효력이 발생합니다.

② 개발자는 「약관의 규제에 관한 법률」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 등 관련 법령을 위배하지 않는 범위에서 본 약관을 변경할 수 있습니다.

③ 개발자가 약관을 변경하고자 하는 경우, 변경 내용과 적용일자를 명시하여 적용일로부터 최소 10일 전에 앱 내 공지 또는 웹페이지를 통해 고지합니다. 다만, 이용자에게 불리한 내용으로 변경하는 경우에는 적용일로부터 최소 30일 전에 고지합니다.

④ 이용자가 변경된 약관의 적용일 이후에도 서비스를 계속 이용하는 경우, 변경된 약관에 동의한 것으로 봅니다. 변경된 약관에 동의하지 않는 이용자는 앱을 삭제하여 이용을 중단할 수 있습니다.

제3조 (서비스의 내용)

① 개발자가 제공하는 서비스의 내용은 다음과 같습니다.

  1. 이용자가 선택한 앱의 알림을 수집·분석하여 AI 기반 브리핑을 생성하는 기능
  2. 자연어 텍스트 또는 음성 입력을 통한 앱 내 데이터 조회·관리 명령 처리 기능
  3. 인간관계(멤버) 관리, 대화방 관리, 개인 상태 기록 등 데이터 관리 기능
  4. 기타 개발자가 추가로 개발하여 제공하는 기능

② 본 서비스는 무료로 제공됩니다. 다만, AI 기능 이용에 필요한 API 키의 발급 및 이에 따른 API 사용료는 이용자가 Google에 직접 부담하며, 개발자는 이에 관여하지 않습니다.

③ 본 서비스를 이용하기 위해서는 Android 운영체제가 탑재된 모바일 기기, 인터넷 연결, 그리고 AI 기능 이용 시 유효한 Gemini API 키가 필요합니다.

제4조 (서비스의 변경 및 중단)

① 개발자는 운영상, 기술상 필요에 따라 서비스의 전부 또는 일부를 변경하거나 중단할 수 있습니다.

② 서비스의 변경 또는 중단이 예정된 경우, 개발자는 변경·중단 사유 및 일정을 사전에 앱 내 공지 또는 웹페이지를 통해 고지합니다. 다만, 개발자가 통제할 수 없는 사유(천재지변, 외부 API 서비스의 장애·변경·중단, 기기 호환성 문제 등)로 인한 경우에는 사전 고지 없이 서비스가 변경되거나 중단될 수 있습니다.

③ 다음 각 호의 사유로 서비스가 정상적으로 제공되지 않을 수 있으며, 이에 대해 개발자는 책임을 지지 않습니다.

  1. Google Gemini API의 장애, 변경, 사용 정책 변경 또는 서비스 종료
  2. AI 모델의 업데이트·변경으로 인한 응답 품질의 변화
  3. 이용자의 기기 환경, 운영체제 업데이트, 네트워크 상태에 따른 서비스 이용 제한
  4. 기타 개발자의 귀책사유가 아닌 외부 요인

④ 개발자는 서비스의 변경 또는 중단으로 인하여 발생하는 문제에 대해, 개발자의 고의 또는 중대한 과실이 없는 한 책임을 지지 않습니다.

제5조 (이용자의 의무)

① 이용자는 서비스 이용 시 다음 각 호의 행위를 하여서는 안 됩니다.

  1. 본 앱을 이용하여 법령 또는 공서양속에 반하는 행위
  2. 본 앱의 기능을 악용하여 타인의 권리를 침해하거나 타인에게 피해를 주는 행위
  3. 본 앱을 역설계(reverse engineering), 디컴파일, 디스어셈블하거나 기타 방법으로 소스 코드를 추출하려는 행위
  4. 본 앱의 정상적인 운영을 방해하거나 보안을 침해하는 행위
  5. 본 앱을 통해 수집·생성된 타인의 정보를 본 앱의 서비스 목적 외로 이용하거나 제3자에게 제공하는 행위
  6. 기타 관련 법령에 위반되는 행위

② 이용자가 제1항의 의무를 위반하여 개발자 또는 제3자에게 손해를 발생시킨 경우, 이용자는 그 손해를 배상할 책임이 있습니다.

제6조 (API 키 관리 책임)

① AI 기능을 이용하기 위해 필요한 API 키는 이용자가 Google AI Studio 등을 통해 직접 발급받아야 하며, API 키의 발급·관리·갱신·폐기에 관한 모든 책임은 이용자에게 있습니다.

② 본 앱은 이용자의 API 키를 기기 내 암호화된 저장소에 보관하며, 개발자의 서버로 전송하거나 별도로 수집하지 않습니다.

③ API 키의 유출, 도용, 무단 사용 등으로 인해 발생하는 모든 문제(API 사용료 과다 청구 포함)에 대해 개발자는 책임을 지지 않습니다.

④ 이용자는 API 키가 유출되었거나 무단으로 사용된 것을 인지한 경우, 즉시 Google AI Studio에서 해당 키를 폐기하고 새로운 키를 발급받아야 합니다.

⑤ API 키 사용에 따른 Google의 이용약관, 사용 정책 및 과금 정책은 Google이 별도로 정하는 바에 따르며, 개발자는 이에 관여하지 않습니다.

중요: API 키는 이용자의 Google 계정과 연결된 인증 정보입니다. 타인과 공유하지 마시고, 유출이 의심되면 즉시 폐기하시기 바랍니다.

제7조 (타인의 정보에 관한 의무)

① 본 앱은 이용자가 알림 접근 권한을 허용한 경우, 메신저 앱 등에서 수신되는 타인의 메시지를 포함한 알림 데이터를 수집·분석합니다. 이용자는 이러한 알림 데이터에 타인의 개인정보가 포함될 수 있음을 인지하고, 다음 각 호의 의무를 준수하여야 합니다.

  1. 본 앱을 통해 수집·열람한 타인의 메시지, 대화 내용 등을 본 앱의 서비스 이용 목적 외로 사용하지 않을 것
  2. 타인의 개인정보를 본 앱 외부로 복사·전송·공개하거나 제3자에게 제공하지 않을 것
  3. 타인의 메시지를 이용하여 해당 타인을 감시·추적하거나 스토킹 등 불법적인 목적으로 사용하지 않을 것
  4. 타인의 개인정보가 포함된 데이터를 이용자의 관리 소홀로 유출되지 않도록 기기의 보안을 유지할 것

② 이용자가 제1항의 의무를 위반하여 타인의 권리를 침해하거나 관련 법령을 위반한 경우, 그로 인한 모든 법적 책임은 이용자에게 있으며 개발자는 이에 대해 책임을 지지 않습니다.

③ 개발자는 이용자의 알림 데이터를 개발자의 서버에 수집하거나 저장하지 않습니다. 알림 데이터는 이용자의 기기 내에서만 처리·저장됩니다.

제8조 (AI 서비스 면책)

중요: AI가 생성한 브리핑, 요약, 분류, 중요도 판정 등의 결과는 참고 목적으로만 제공되며, 그 정확성·완전성·적시성을 보장하지 않습니다.

① 본 앱의 AI 기능은 외부 AI 서비스(Google Gemini)를 활용하여 알림 메시지를 분석·요약하고 자연어 명령을 처리합니다. AI가 생성한 모든 결과물(브리핑, 사건 요약, 중요도 판정, 카테고리 분류, 명령 실행 결과 등)은 참고 정보로서 제공되며, 그 정확성, 완전성 또는 적시성을 보장하지 않습니다.

② AI 기능의 한계로 인해 다음과 같은 문제가 발생할 수 있으며, 이에 대해 개발자는 책임을 지지 않습니다.

  1. 중요하거나 긴급한 메시지를 누락하거나 중요도를 낮게 판정하는 경우
  2. 메시지 내용을 부정확하게 요약하거나 맥락을 잘못 해석하는 경우
  3. 자연어 명령을 잘못 이해하여 의도와 다른 동작을 수행하는 경우
  4. AI 모델의 특성상 환각(hallucination) 현상으로 사실과 다른 내용을 생성하는 경우
  5. 네트워크 장애, API 오류 등으로 AI 기능이 정상 작동하지 않는 경우

③ 이용자는 AI가 생성한 결과를 최종 판단의 근거로 사용하여서는 안 되며, 중요한 의사결정은 반드시 원본 메시지를 직접 확인한 후 수행하여야 합니다.

④ 이용자가 AI 결과를 과신하여 발생한 직접적·간접적 손해(긴급 메시지 미확인, 잘못된 정보에 기반한 행동 등)에 대해 개발자는 책임을 지지 않습니다.

제9조 (일반 면책사항)

① 개발자는 다음 각 호의 사유로 발생한 손해에 대해 책임을 지지 않습니다.

  1. 이용자의 귀책사유로 인한 서비스 이용 장애 또는 데이터 손실
  2. 이용자가 기기 분실, 도난, 앱 데이터 초기화 등으로 인해 데이터를 상실한 경우
  3. 이용자의 기기 환경(운영체제 버전, 설정, 다른 앱과의 충돌 등)으로 인해 서비스가 정상 작동하지 않는 경우
  4. 무료로 제공되는 서비스의 이용과 관련하여 「약관의 규제에 관한 법률」 등 관련 법령에 반하지 않는 범위 내의 손해

② 본 앱은 이용자의 데이터를 기기 내에만 저장하며, 별도의 서버 백업을 제공하지 않습니다. 따라서 기기 변경, 초기화, 앱 삭제 등으로 인한 데이터 손실에 대해 개발자는 복구 의무를 부담하지 않습니다.

③ 전항의 면책사항에도 불구하고, 관련 법령에 따라 개발자가 면책될 수 없는 사항에 대해서는 해당 법령이 정하는 바에 따릅니다.

제10조 (지적재산권)

① 본 앱의 소프트웨어, 디자인, 로고, 아이콘, 레이아웃, 텍스트, 소스 코드 등 서비스를 구성하는 모든 저작물에 대한 지적재산권은 개발자에게 귀속됩니다.

② 이용자는 개발자의 사전 서면 동의 없이 본 앱의 전부 또는 일부를 복제, 배포, 방송, 전시, 전송, 2차적 저작물 작성 등의 방법으로 이용하거나 제3자에게 이용하게 하여서는 안 됩니다.

③ 이용자가 서비스를 이용하는 과정에서 직접 입력하거나 생성한 데이터(메모, 상태 기록, 피드백 등)에 대한 권리는 이용자에게 귀속됩니다.

④ AI 기능을 통해 생성된 결과물(브리핑, 요약, 사건 카드 등)의 지적재산권에 대해서는 해당 AI 서비스 제공자(Google)의 이용약관에서 정하는 바에 따릅니다.

제11조 (손해배상)

① 개발자의 고의 또는 중대한 과실로 인하여 이용자에게 손해가 발생한 경우, 개발자는 관련 법령에 따라 그 손해를 배상할 책임이 있습니다.

② 이용자가 본 약관에서 정한 의무를 위반하여 개발자 또는 제3자에게 손해를 발생시킨 경우, 이용자는 그 손해를 배상할 책임이 있습니다.

③ 본 앱은 무료로 제공되는 서비스이므로, 개발자의 고의 또는 중대한 과실이 없는 한 서비스 이용과 관련하여 발생하는 손해에 대한 개발자의 배상 책임은 관련 법령이 허용하는 최대 범위 내에서 제한됩니다.

제12조 (분쟁 해결)

① 본 약관에 관한 분쟁은 대한민국 법률을 준거법으로 합니다.

② 서비스 이용과 관련하여 개발자와 이용자 간에 분쟁이 발생한 경우, 양 당사자는 분쟁의 해결을 위해 성실히 협의합니다.
